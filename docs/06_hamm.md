# Counting Point Mutations

## [Rosalind Problem](http://rosalind.info/problems/hamm/){:target="_blank"}

**Given**: Two DNA strings $s$ and $t$ of equal length (not exceeding 1 kbp).

**Return**: The Hamming distance $d_H(s,t)$.

### Sample Dataset

```
GAGCCTACTAACGGGAT
CATCGTAATGACGGCCT
```

### Sample Output

```
7
```
