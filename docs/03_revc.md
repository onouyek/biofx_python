# Complementing a Strand of DNA

## [Rosalind Problem](https://rosalind.info/problems/revc/){:target="_blank"}

**Given**: A DNA string s of length at most 1000 bp.

**Return**: The reverse complement $s^c$ of s.

### Sample Dataset

```
AAAACCCGGT
```

### Sample Output

```
ACCGGGTTTT
```
